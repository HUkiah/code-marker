场景演示

发布者使用了 Code Marker 组件

此场景下：发布者在帖子中嵌入了组件 → 所有读者（无论是否装了油猴脚本）都能交互标记

[分享] AI 写作工具 30 天试用邀请码 x6

刚拿到一批内测邀请码，每人限用一个，注册时填入即可激活。用完请标记一下帮助后面的朋友：

提示：用完后点「已用」，验证可用点「可用」，帮助他人节省时间。

工作原理：
1. 组件挂载后设置 data-cm-active="true"
2. 如果用户安装了油猴脚本，脚本检测到此标志后自动退让
3. 标记数据存储在后端 API（当前指向 localhost:3456 开发服务器）

发布者只需粘贴：

<div class="code-marker" data-codes="码1,码2,码3" data-page-id="可选唯一标识"></div> <script src="http://localhost:3456/widget.js"></script>
